# Desafio PHP - Vue / Laravel

## Laravel 9 - Vue - Inertia.

prequisitos: php: ^8.1.4 e composer ^2.3.5

```bash
cp .env.example .env
```

Pacotes Laravel
```Bash
composer install
```

Instalando o Vue
```Bash
npm install
```

Banco de Dados

```bash
php artisan migrate
```

Iniciando Laravel

```bash
php artisan serve
```

Iniciando NPM

```bash
npm run dev
```
